Yeh raha ek **Character Counter** project ke liye **HTML aur JavaScript** par based **20-30 step-by-step tasks**. Har task complete karte jaoge toh tumhara project systematically develop hota jayega.  

---

### âœ… **HTML Tasks** (1-10)  
#### Yeh basic structure create karne ke liye hain:

1. Ek simple **HTML file** banao.  
2. Ek **textarea** element add karo jisme user text likh sake.  
3. Ek **div** ya `p` element banao jo character count show karega.  
4. Ek **button** add karo jo textarea ko clear karega.  
5. Ek **heading** add karo jo project ka naam dikhaye (`Character Counter`).  
6. Ek **label** add karo jo textarea ke upar ho (`Enter your text:`).  
7. Ek **maximum character limit** define karne ke liye ek `span` element banao.  
8. Ek **progress bar** ka basic structure banao jo max limit tak pahunchne par fill ho.  
9. Ek **container div** banao jo sab elements ko center mein rakhe.  
10. Ek **footer** add karo jisme tumhara naam ya project details ho.

---

### âœ… **JavaScript Tasks** (11-30)  
#### Yeh tasks project ko functional aur interactive banayenge:

11. JavaScript file ko HTML ke saath connect karo (`script.js`).  
12. Textarea ke andar likhne par **character count update** karne ka function likho.  
13. Jo count show ho raha hai usko dynamically **innerText** se update karo.  
14. Ek **maximum limit** set karo (e.g., 200 characters) . 
15. Agar user max limit cross kare toh **red color** se warning dikhaye.  
16. Agar limit cross ho jaye toh extra characters **disable** kar do.  
17. Clear button pe click karne par textarea ko **empty** karo.  
18. Clear button click hone par **counter bhi reset** ho.  
19. Agar textarea khali ho toh counter **0 dikhaye**.  
20. Ek **progress bar fill** hone ka logic likho (characters likhne par increase ho).  
21. Progress bar ka **background color change** ho jab max limit ke pass aaye.  
22. Jo warning text dikh raha ho usko **smooth transition** se dikhaye.  
23. Ek function likho jo textarea me spaces bhi count kare ya ignore kare (settings optional).  
24. Jo text likha hai usko **localStorage me save** karo taaki refresh hone pe na gayab ho.  
25. Page load hone par **saved text automatically load** ho jaye.  
26. Ek **toggle switch** add karo jo character counting mode change kare (`With Spaces` / `Without Spaces`).  
27. Ek **word counter** bhi add karo jo words count kare.  
28. Ek **copy button** banao jo textarea ka content copy kare clipboard me.  
29. Ek **dark mode** toggle banao jo UI theme change kare.  
30. Jo bhi CSS animations aur transitions use kiye hain, unko JS se dynamically handle karne ka function likho.
